# أساسيات الشبكات

## ما هي الشبكة؟
هي ربط أجهزة مع بعض لتبادل البيانات.

## أنواع الشبكات:
- LAN (شبكة محلية)
- WAN (شبكة واسعة)

## أهم المصطلحات:

### IP Address:
عنوان الجهاز داخل الشبكة

### DNS:
تحويل اسم الموقع إلى IP

### Router:
جهاز يربط الشبكات ببعض

## مثال عملي:
إذا الإنترنت ما يشتغل:
- تأكد من الاتصال بالشبكة
- تحقق من إعدادات IP
- أعد تشغيل الراوتر
